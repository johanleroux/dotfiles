# Backup Checklist Before OS Installation

## Documents & Personal Files
- [ ] Documents folder
- [ ] Desktop files
- [ ] Downloads folder
- [ ] Pictures and photos
- [ ] Videos
- [ ] Music library

## Development & Configuration
- [ ] SSH keys (`~/.ssh/`)
- [ ] GPG keys
- [ ] AWS keys
- [ ] Git configuration (`~/.gitconfig`)
- [ ] IDE settings and plugins
- [ ] Custom scripts and aliases
- [ ] Environment variables (`.bashrc`, `.zshrc`, etc.)

## Application Data
- [ ] Browser bookmarks and extensions
- [ ] Email clients data
- [ ] Password manager databases
- [ ] Application preferences
- [ ] License keys and activation codes

## System Configuration
- [ ] Network settings (WiFi passwords)
- [ ] Printer configurations
- [ ] Custom fonts
- [ ] Hosts file modifications
- [ ] Cron jobs or scheduled tasks

## Database & Projects
- [ ] Active project folders
- [ ] Database dumps
- [ ] Virtual machines
- [ ] Docker volumes and images

## Cloud & Remote
- [ ] Verify cloud sync status
- [ ] Remote repository pushes
- [ ] Backup encryption keys

## Notes
- Store backups on external drive or cloud storage
- Verify backup integrity before proceeding
- Document installed software for reinstallation
